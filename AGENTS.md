# AGENTS.md

This repository is a public GitHub profile repository.

## Source Of Truth

- `README.md`: public profile content.
- `docs/profile-source.md`: source facts used to generate the profile.

## Operating Rules

- Keep claims factual and traceable to CV, LinkedIn, or public GitHub evidence.
- Do not feature old, small, or forked repositories as proof of current production ability.
- Do not publish phone numbers or sensitive personal data in public Markdown.
- Keep edits focused on the profile and supporting documentation.

## Quality Checks

- Run `make check` before publishing changes.
- Keep Markdown links valid when they point to local files.
